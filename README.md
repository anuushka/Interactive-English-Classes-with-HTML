# interactive-HTML-classes

HTML for English morning/evening classes and flashcard

## **Template 1 for lectures from now on:**

| **Placeholder**              | **What to replace**                              |
|----------------------------|--------------------------------------------------|
| `[DAY]`                    | Monday / Tuesday etc.                           |
| `[BOOK] / [CHAPTER] / [LEVEL]` | Good to Great · Ch.1 · B1                  |
| `[GRAMMAR FOCUS]`          | The grammar topic + structures                  |
| `Tab label R / Reading`    | Change to `L / Listening` on odd days           |
| `[VOCAB × 5]`              | Pre-taught words + Mongolian                    |
| `[HOOK]`                   | The opening personal question                   |
| `[QUOTE] + [DATA TABLE]`   | Key quote + numbers from the text               |
| `[PROMPT]`                 | The debate or speaking prompt                   |
| `[SIDE A] / [SIDE B]`      | Debate positions                               |
| `[CLOSING SENTENCE]`       | The stem for the closing round                  |
| `[SOS PROMPTS]`            | 3–4 content-specific emergency questions        |
